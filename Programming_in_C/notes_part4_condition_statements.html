Programming in C - Part 4

condition statements

If Statement

The if statement is used to execute a block of code only when a given condition is true.

#include <stdio.h>

int main() {
  int age = 20;

  if (age >= 18) {
    printf("You are an adult.\\n");
  }

  return 0;
}

You are an adult.

If-Else Statement

The if-else statement allows you to choose between two blocks of code depending on whether the condition is true or false.

#include <stdio.h>

int main() {
  int number = 7;

  if (number % 2 == 0) {
    printf("The number is even.\\n");
  } else {
    printf("The number is odd.\\n");
  }

  return 0;
}

The number is odd.

If-Else Ladder

An if-else ladder is used when you need to check multiple conditions one after another.

#include <stdio.h>

int main() {
  int marks = 75;

  if (marks >= 90) {
    printf("Grade: A\\n");
  } else if (marks >= 75) {
    printf("Grade: B\\n");
  } else if (marks >= 50) {
    printf("Grade: C\\n");
  } else {
    printf("Fail\\n");
  }

  return 0;
}

Grade: B

Switch Statement

The switch statement is used when you have multiple choices for a single variable.

#include <stdio.h>

int main() {
  int day = 3;

  switch (day) {
    case 1:
      printf("Monday\\n");
      break;
    case 2:
      printf("Tuesday\\n");
      break;
    case 3:
      printf("Wednesday\\n");
      break;
    default:
      printf("Invalid day\\n");
  }

  return 0;
}

Wednesday
