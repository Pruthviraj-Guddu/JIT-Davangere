Module 1: Introduction to C

Introduction to Computers

A computer is an electronic device which accepts input from the user in the form of data and instructions, processes it, and produces output.

  • Input: Data and commands sent to the computer.
  • Processing: Operations performed by hardware and software.
  • Output: The result produced and displayed.
  • Storage: Saving results inside or outside the computer.

Characteristics of Computers

  • Speed: Perform millions of operations per second.
  • Accuracy: Accurate results if inputs are correct. (Garbage In, Garbage Out).
  • Automation: Perform tasks without human intervention.
  • Diligence: Do not get tired of repetitive tasks.
  • Versatility: Perform multiple different tasks simultaneously.
  • Memory: Internal (primary) and external (secondary) storage.
  • No IQ: No decision-making ability without instructions.
  • Economical: Save time, money, and energy.

Data vs Information

Data: Raw facts (e.g., date of birth).

Information: Processed data (e.g., calculating age from date of birth).

Introduction to C

  • Structure of a C Program: Includes headers, main function, and statements.
  • Files used: Source code (.c), headers (.h), object files, executables.
  • Compiler: Converts source code into machine code.
  • Execution: Compile → Link → Run.

History of Computers

The history of computers is divided into five generations, each based on the technology used for processing and storage. Below are the details of each generation along with their advantages, disadvantages, and examples.

First Generation (1942–1955): Vacuum Tubes

  • Built using thousands of vacuum tubes as the main technology.
  • Electromagnetic relays were used as primary memory.
  • Punch cards were used to store data and instructions.
  • Programming was done in machine language and assembly language.

Advantages: Fastest calculating devices of their time.

Disadvantages: Generated excessive heat, consumed a lot of electricity, were bulky, required constant maintenance, and were very expensive.

Example: ENIAC (Electronic Numerical Integrator and Computer).

Second Generation (1955–1964): Transistors

  • Used transistors, which were smaller, cheaper, faster, and more reliable than vacuum tubes.
  • Magnetic core memory was used as primary memory.
  • Magnetic tapes and disks were used for storage.
  • Programming was done in assembly language and early high-level languages like FORTRAN and COBOL.

Advantages: Consumed less electricity, produced less heat, faster and more reliable, supported high-level programming.

Disadvantages: Manual assembly of transistors made production expensive and complex.

Example: IBM 1401, IBM 7090.

Third Generation (1964–1975): Integrated Circuits (ICs)

  • Used Integrated Circuits (ICs), combining transistors, capacitors, and resistors on a single chip.
  • Larger magnetic core memory was used as primary memory.
  • Magnetic tapes and disks were used for secondary storage.
  • Programming was done in high-level languages like FORTRAN, COBOL, and BASIC.

Advantages: Could perform a million instructions per second, smaller, cheaper, and more reliable, supported scientific and business applications.

Disadvantages: Still generated heat and required cooling systems.

Example: IBM 360 Series, PDP-8.

Fourth Generation (1975–1989): Microprocessors

  • Used LSI (Large Scale Integration) and later VLSI (Very Large Scale Integration) microprocessor chips.
  • Semiconductor memory was used as primary memory.
  • Floppy disks and magnetic tapes were used as portable storage devices.
  • Programming was done in C and later C++.

Advantages: Smaller, cheaper, faster, more reliable, required less electricity, supported graphical user interfaces (GUIs), widely used in offices and homes.

Disadvantages: Still not intelligent systems; required manual programming.

Example: Intel 8086, Apple Macintosh, IBM PC.

Fifth Generation (1989–Present): Artificial Intelligence

  • Uses ULSI (Ultra Large Scale Integration) chips.
  • Semiconductor memory is used as primary storage.
  • Programming is done in modern languages like Java, Python, C#, and R.
  • Focused on Artificial Intelligence (AI), Machine Learning, Natural Language Processing, and Cloud Computing.

Advantages: Extremely powerful, portable, reliable, consume less power, support advanced features like AI, robotics, and expert systems.

Disadvantages: Expensive to build and maintain, still evolving with ethical and social challenges.

Examples: Modern laptops, smartphones, AI-powered systems like Google Assistant, Siri, and self-driving cars.

Classification of Computers

Computers are classified based on their size, speed, cost, and purpose. The major types are Supercomputers, Mainframes, Minicomputers, Microcomputers, Smartphones, and Tablets. Below is a detailed explanation with examples.

1) Supercomputers

  • Fastest, most powerful, and most expensive computers.
  • First developed in the 1980s to solve complex scientific problems and process massive data sets.
  • Use parallel processing and can perform trillions of calculations per second.
  • Support thousands of users simultaneously.
  • Applications: Weather forecasting, nuclear research, aircraft design, automotive design, online banking.

Examples: CRAY-1, CRAY-2, ETA A-10.

2) Mainframe Computers

  • Large-scale computers, smaller than supercomputers but still very powerful.
  • Very expensive and require large air-conditioned rooms.
  • Support multiple processors and thousands of simultaneous users.
  • Terminals:
    • Dumb terminals: Only monitor and keyboard, rely on mainframe CPU and storage.
    • Intelligent terminals: Have their own processor but no storage.
  • Applications: Used in banks, airlines, universities, and large organizations where many users need data access.

Examples: IBM S/390, Control Data CYBER 176, Amdahl 580.

3) Minicomputers (Midrange Computers)

  • Smaller, cheaper, and slower than mainframes.
  • Bridge the gap between mainframes and personal computers.
  • Can be used by single users for complex design tasks or multiple users simultaneously.
  • Applications: Business, education, hospitals, and government organizations.
  • The first minicomputer was introduced by Digital Equipment Corporation in the 1960s.

4) Microcomputers (Personal Computers)

The most common type of computer, designed for individual use. First introduced by IBM in 1981 as IBM-PC.

  • Desktop PCs: The most popular model for offices and homes.
  • Laptops: Portable, with similar storage and performance to desktops. Have built-in keyboard and touchpad.
  • Workstations: High-performance PCs with powerful processors, high-resolution monitors, and large memory. Used for engineering, video editing, and design.
  • Network Computers: Lightweight machines designed for network use, often with little or no storage. Examples include Web TV and Internet PCs.

Advantages of Workstations: High resolution, powerful for animations and engineering design.

Disadvantages: Expensive and not portable.

5) Smartphones

  • Handheld electronic devices combining mobile phone and PDA features.
  • Support internet browsing, email, SMS, voice recognition, and multimedia.
  • Applications: Communication, entertainment, productivity, and business.

Advantages: Multi-functional, internet connectivity, PC compatibility.

Disadvantages: More expensive, larger than basic phones, high replacement cost.

6) Tablet Computers

  • Portable PCs shaped like a slate, with touch screen and stylus support.
  • Can rotate display by 180° and perform all functions of a laptop.
  • Applications: Digital note-taking, design, education, business presentations.

Advantages: Lightweight, portable, paperless workflow.

Disadvantages: Costly, delicate, high replacement cost.

Basic Organization of Computers

A computer is an electronic device that performs five major operations: Input, Storage, Processing, Output, and Control. These operations are coordinated by the CPU and supported by input/output devices and memory units.

Input

Input is the process of entering data and instructions into a computer system. Various devices can be used to provide input:

  • Keyboard: Standard and numeric keys for text and number entry.
  • Mouse, Trackball, Touchpad: Pointing devices to control cursor movement.
  • Scanner, Barcode Reader, OCR, OMR: Convert physical data into digital format.
  • Microphone and Camera: For audio and visual input.

Input devices convert user data into binary code (0s and 1s), which the computer can understand.

Storage

Storage is the process of saving data and instructions for processing. Computers have two main types of storage:

  • Primary Storage (Main Memory): Directly accessible by the CPU at high speed. Stores data, programs, and intermediate results. Example: RAM. Volatile: Data is lost when the computer is turned off.
  • Secondary Storage (Auxiliary Memory): Permanent storage for data and programs not currently in use. Cheaper and non-volatile. Example: Hard Disk Drives (C: and D: drives), SSDs.

Processing

Processing is the act of performing operations on data according to instructions provided by the user program. Data moves between main memory and the Arithmetic Logic Unit (ALU) multiple times until processing is complete.

Output

Output is the process of presenting processed data to the user in a readable form. Output devices convert binary results from the CPU into human-readable format.

  • Monitor: Displays text, images, and videos.
  • Printer: Produces hard copies of documents.
  • Speakers: Output for audio signals.

Control

The Control Unit (CU) acts like the nervous system of the computer, coordinating and managing all components. CPU = CU + ALU. The ALU performs calculations and logic operations, while the CU activates and monitors input, output, and memory.

Inside the Computer

  • CPU: The brain of the computer, performing all calculations and coordinating other devices. Faster CPU = quicker program execution.
  • RAM: Temporary memory used by currently running programs. More RAM allows multiple applications to run smoothly.
  • Hard Disk Drive (HDD): Permanent storage for data, documents, and programs. Larger drives store more information.
  • Video Card: Generates images for display. High-end cards have their own RAM and processors for multimedia or gaming.
  • Sound Card: Enables audio output and input for speakers and microphones.
  • Modem: Connects the computer to the Internet via telephone lines.
  • Network Card: Connects the computer to other computers or networks for data transfer.
  • Fans & Cables: Keep the system cool and provide power/communication between components.

Input Devices

Input devices allow users to interact with the computer. They can be categorized as:

  • Keyboard: Main input device for text and numbers. Advantages: Easy and inexpensive. Disadvantages: Cannot draw; cursor movement is slow.
  • Pointing Devices: Mouse, trackball, joystick, touchpad. Used to control cursor, select commands, and draw graphics. Faster and more intuitive than keyboard for graphical operations.
  • Other Devices: Scanners, OCR, OMR, barcode readers, cameras, microphones for multimedia input.
